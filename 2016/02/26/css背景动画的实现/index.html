css背景动画的实现

  • css
  • html
  • js
  • web技术
  • 动画

最近写了一下按想法写了一些html的动画效果发现一个问题!
jQuery是不支持动画背景颜色和文字颜色动画的,虽然能修改css属性来实现修改背景颜色,但是我想要的效果是动画效果!
于是乎我在网络间游荡中获得了一个简单的处理方案!

方案到处有在哪找到的忘记了,我都是这样找就找到没必要去记得出自谁手,如果您发现是您的作品,那么我在这里先非常感谢了!
如果在我这里拿的原创就随便啦,反正我不看也不用谢,分享快乐,对你有用即可!
这个端代码是高人从jqurey UI里面提取的一段可以用作背景颜色动画的代码个人非常喜欢,于是也用上了自己的站点,虽然做得还有些土,以后有新想法再改吧!
废话不多说上代码
复制保存成JS文件即可!
原名叫

jquery.colorAnimations.js
最好自己拿去压缩一下,跟方便引入!至于color的动画方法还在找,找到了再分享!
